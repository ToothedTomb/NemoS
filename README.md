# NemoS - Text Editor For Linux:

NemoS - Is a simple Terminal Text Editor that is based on Nano. 

# Instuctions:

 Arrow Keys: Navigate /n
 Enter: Insert new line /n
 Backspace: Delete character /n 
 Ctrl+S: Save file /n
 Ctrl+R: Rename file
/n
 Ctrl+X: Exit editor
