# NemoS - Text Editor For Linux:

NemoS - Is a simple Terminal Text Editor that is based on Nano. 
# Instuctions:

 Arrow Keys: Navigate 
 Enter: Insert new line 
 Backspace: Delete character 
 Ctrl+S: Save file 
 Ctrl+R: Rename file
 Ctrl+X: Exit editor
